# 김도균 개인 홈페이지 프로젝트

## 개요

이 프로젝트는 김도균의 개인 홈페이지를 구축하는 것을 목표로 합니다. 최신 웹 기술과 현대적인 디자인을 활용하여 김도균의 경력, 프로젝트, 관심사 등을 소개하는 매력적인 공간을 만듭니다.

**GitHub Repository:** [https://github.com/kdg015b/exe-1.git](https://github.com/kdg015b/exe-1.git)

## 디자인 및 기능

*   **디자인:**
    *   모던하고 깔끔한 디자인
    *   반응형 레이아웃으로 모든 기기에서 최적화된 경험 제공
    *   인상적인 타이포그래피와 색상 팔레트 사용
    *   애니메이션과 상호작용 요소를 통해 동적인 사용자 경험 제공
*   **주요 기능:**
    *   **자기소개:** 김도균에 대한 간략한 소개
    *   **포트폴리오:** 진행했던 프로젝트들을 소개하고 관련 링크 제공
    *   **기술 스택:** 사용 가능한 기술 및 도구 목록
    *   **경력:** 학력 및 경력 사항
    *   **블로그 (선택 사항):** 개인적인 생각이나 기술적인 지식을 공유하는 공간
    *   **문의하기:** 방문자가 연락을 남길 수 있는 폼

## 현재 작업 계획

### 1단계: 기본 구조 설정 (완료)
1.  **기본적인 HTML, CSS, JavaScript 파일 생성:** 프로젝트의 기본 구조를 설정합니다.
2.  **`index.html`:** "안녕하세요, 김도균입니다."라는 환영 메시지를 포함한 기본 HTML 구조를 작성합니다.
3.  **`style.css`:** 기본 스타일링을 추가하여 페이지의 전반적인 디자인을 설정합니다.
4.  **`main.js`:** 페이지의 동적인 기능을 위한 기본 JavaScript 파일을 작성합니다.
5.  **HTML에 CSS 및 JavaScript 연결:** 생성된 CSS와 JavaScript 파일을 `index.html`에 연결합니다.

### 2단계: 주요 섹션 추가
1.  `index.html`에 **자기소개(About)**, **포트폴리오(Portfolio)**, **기술 스택(Skills)** 섹션을 추가합니다.
2.  각 섹션에 대한 기본 콘텐츠를 채웁니다.
3.  `style.css` 파일을 수정하여 추가된 섹션의 레이아웃과 스타일을 지정합니다.
