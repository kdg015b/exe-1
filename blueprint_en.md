# Do-Gyun Kim's Personal Website Project

## Overview

This project aims to build a personal website for Do-Gyun Kim. Utilizing the latest web technologies and modern design, it will create an engaging space to introduce Do-Gyun Kim's career, projects, and interests.

**GitHub Repository:** [https://github.com/kdg015b/exe-1.git](https://github.com/kdg015b/exe-1.git)

## Design and Features

*   **Design:**
    *   Modern and clean design
    *   Responsive layout providing an optimized experience on all devices
    *   Use of impressive typography and color palettes
    *   Dynamic user experience through animations and interactive elements
*   **Key Features:**
    *   **About Me:** A brief introduction to Do-Gyun Kim
    *   **Portfolio:** Introduce past projects and provide relevant links
    *   **Skills:** A list of usable technologies and tools
    *   **Experience:** Academic and professional history
    *   **Blog (Optional):** A space to share personal thoughts or technical knowledge
    *   **Contact:** A form for visitors to leave messages
